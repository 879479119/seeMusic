# See Music

> 使用vue vuex webpack Electron搭建的跨平台云音乐播放器，网易云音乐接口强力支撑，可随意畅听收费歌曲


## 使用

``` bash
# 项目根目录安装依赖
npm install

# 浏览器访问localhost:8080
npm run dev

# 打包编译生成静态资源
npm run build

# 测试Electron应用
npm run test
```

## 跨平台开发

```bash
# 在dist目录下安装electron相关依赖
npm run install

# 编译生成可执行文件
npm run build
```
